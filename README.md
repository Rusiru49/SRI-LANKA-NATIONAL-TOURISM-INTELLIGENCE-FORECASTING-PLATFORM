# üá±üá∞ Sri Lanka National Tourism Intelligence & Forecasting Platform

An end-to-end **data-driven tourism analytics and forecasting platform** designed to support **policy makers, researchers, and tourism stakeholders** in Sri Lanka. This system integrates **data collection, preprocessing, visualization, and AI-based forecasting** to generate actionable insights on tourism trends.

---

## üöÄ Project Overview

The **Sri Lanka National Tourism Intelligence & Forecasting Platform** leverages historical tourism data to:

* Analyze tourism performance across **years, regions, and seasons**
* Visualize trends using **interactive dashboards**
* Forecast future tourist arrivals using **machine learning models**
* Support **evidence-based decision making** in tourism planning

This project was developed as a **Data Science academic project** with a strong focus on **real-world applicability**.
